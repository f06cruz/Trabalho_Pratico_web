# Trabalho_Pratico_web
Um website front-end completo, funcional e visualmente consistente, aplicando os conhecimentos adquiridos  ao longo da unidade Programação WEB.

Nome do projeto: 
Inês Cruz - Tradutora Freelance. 
Tema escolhido: 
Website de tradutora freelancer para apresentação da mesma a prospetivos clientes. 
Tecnologias utilizadas: 
html, css, bootstap, javascript. 
Funcionalidades principais: 
página de apresentação, 
portfolio á base de tabs, 
grupo co-criado pela tradutora apresentado atraves de acordião, 
informaçâo sobre a tradutora (apenas containers), 
contacto com a tradutora por links ou envio de email através de formulário (com verificação).
Eventuais limitações ou ideias futuras: 
melhorar a página de contacto (procurar uma serviço melhor para o envio do email), 
criar butão para mudar as páginas para versão inglesa.
