# Trabalho_Pratico_web
Um website front-end completo, funcional e visualmente consistente, aplicando os conhecimentos adquiridos  ao longo da unidade Programação WEB.

Nome do projeto: Inês Cruz - Tradutora Freelance. 
Tema escolhido: Website de tradutora freelancer para apresentação da mesma a prospetivos clientes. 
Tecnologias utilizadas: html, css, bootstap, javascript. 
Funcionalidades principais: Envio de email através de formulário, 
Eventuais limitações ou ideias futuras: usar o bootstrap para transformar o menu num botão hamburger só em mobile,melhorar a página de contacto (procurar uma serviço melhor para o envio do email), criar butão e páginas em inglês,
